# Ejercicio practico Python API SQLAlchemy

## Instrucciones

Clonar el repositorio `https://github.com/edwinmoreno77/SQLAlchemy_API.git`

- cambiar a la carpeta `cd SQLAlchemy_API`
- pipenv shell
- pipenv install
- pipenv db init
- pipenv db migrate
- pipenv db upgrade
- python app.py
